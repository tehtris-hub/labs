Philosophie

// learn · break · fix · understand

Chaque lab simule un environnement réel avec un problème à résoudre. La VM est fournie dans un état non fonctionnel volontairement simple — l'objectif est de comprendre le pourquoi, pas seulement le comment.

01_

VM Préconfigurée

Une machine virtuelle Ubuntu/Debian vous est fournie. Elle démarre, mais quelque chose ne fonctionne pas. À vous de trouver.

02_

Script Guidé

Un script partiellement rédigé vous guide pas à pas. Les sections critiques sont remplacées par des TODO — à vous de les compléter.

03_

Diagnostic Actif

Avant de corriger, vous devez diagnostiquer. Chaque lab enseigne les commandes de vérification d'état (systemctl, journalctl, ss…).

04_

Solution Dépliable

Bloqué ? La solution complète est disponible dans une section masquée. L'objectif reste d'essayer avant de regarder.

Labs Disponibles

// 10 labs · hands-on

Choisissez un lab et suivez le tutoriel. Chaque exercice correspond à un outil réellement utilisé dans un SOC ou une équipe défensive.

Lab · Linux · Terminal Débutant
Linux 101 — Survivre dans un Terminal
Le terminal vous fait peur ? Normal. Dans ce lab, vous allez apprendre à naviguer, lire des fichiers et tuer des processus en 6 étapes. Votre souris pleurera.
linux bash terminal
Lab · Security · Accounts Débutant
Cédric et le Mot de Passe en Carton
Cédric a mis 'password' comme mot de passe. Oui, vraiment. Votre mission : sauver l'équipe de la honte en forçant un vrai mot de passe.
usermod passwd security
Lab · Crypto · AES Débutant
Le PDF Chiffré du Patron
Votre patron vous a envoyé un document 'ULTRA IMPORTANT' chiffré avec AES. Il vous donne 3 mots de passe possibles. Bonne chance.
openssl aes-256 crypto
Lab · Crypto · GPG Débutant
La Clé Secrète du Patron (GPG)
Vous avez changé d'ordi mais gardé votre clé privée. Le patron a chiffré un message rien que pour vous. Importez la clé, déchiffrez le message, lisez les mauvaises nouvelles.
gpg rsa asymmetric
Lab · Network · Monitoring Débutant
Le Trafic Suspect de la Nuit
Quelque chose envoie des requêtes HTTP vers une IP inconnue toutes les 30 secondes. Utilisez ss et lsof pour trouver le coupable — et l'éliminer.
ss lsof network
Lab · Firewall · iptables Débutant
Jimmy a Cassé le Pare-feu
Jimmy a joué avec iptables et maintenant HTTP est bloqué partout. Jimmy est en vacances. C'est votre problème maintenant.
iptables firewall http
Lab · Forensics · Network Débutant
Qui Écoute sur le Port 4444 ?
Quelqu'un a ouvert un port d'écoute sur votre serveur. Le patron a reçu un message bizarre. Utilisez ss pour trouver l'oreille indiscrète et la fermer.
ss netcat ports
Lab · AV Débutant
Antivirus ClamAV — Installation & Configuration
La VM démarre mais ClamAV n'est pas opérationnel. Base de signatures manquante, service inactif. Corrigez et lancez votre premier scan.
clamav systemd freshclam
Lab · IDS Intermédiaire
Suricata IDS — Interface & Règles
Suricata est installé mais l'interface réseau n'est pas configurée et les règles communautaires sont désactivées. Aucune alerte ne sera générée.
suricata IDS emerging-threats yaml
Lab · Logging Intermédiaire
Sysmon Linux — Télémétrie Système
Sysmon pour Linux est présent mais aucun fichier de configuration XML n'est chargé et le service n'est pas activé. Aucun événement n'est collecté.
sysmon linux syslog xml
