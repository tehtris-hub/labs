ClamAV —
Antivirus Linux

La VM Ubuntu démarre. ClamAV est installé mais inutilisable : la base de signatures est vide et le service est inactif. Votre mission : diagnostiquer, corriger et effectuer votre premier scan.

01 //

Objectif

ClamAV est un antivirus open-source très utilisé sur les serveurs Linux. Il analyse les fichiers à la recherche de malwares connus via une base de signatures régulièrement mise à jour (freshclam).

À la fin de ce lab vous saurez :

  • Vérifier l'état d'un service systemd
  • Mettre à jour la base de signatures ClamAV
  • Lancer un scan manuel avec clamscan
  • Activer le scan en temps réel avec clamd
02 //

État de la VM — Ce qui ne fonctionne pas

Problème détecté

ClamAV est installé (paquet présent) mais : (1) la base de signatures n'a jamais été téléchargée, (2) le service clamd n'est pas activé, (3) freshclam est configuré avec un miroir incorrect.

Commencez par vérifier l'état du système :

Diagnostic initial
# Vérifier si clamav est installé
dpkg -l | grep clamav

# Vérifier l'état du service clamd
systemctl status clamav-daemon

# Vérifier si la base de signatures existe
ls -lh /var/lib/clamav/

# Consulter les logs freshclam
cat /var/log/clamav/freshclam.log
À NOTER Un service systemd a 3 états clés : active (running), inactive (dead), et failed. La commande systemctl status vous indique lequel s'applique.

04 //

Vérification — Premier Scan

Testez votre installation avec le fichier de test EICAR (faux virus reconnu par tous les AV) :

Vérification
# Créer le fichier de test EICAR
echo 'X5O!P%@AP[4\PZX54(P^)7CC)7}$EICAR-STANDARD-ANTIVIRUS-TEST-FILE!$H+H*' > /tmp/eicar.txt

# Scanner le fichier — doit détecter Eicar-Test-Signature
clamscan /tmp/eicar.txt

# Scanner un répertoire entier de façon récursive
clamscan -r --infected /home/

# Vérifier le statut du daemon temps réel
systemctl is-active clamav-daemon && echo "OK: daemon actif"
SUCCÈS ATTENDU clamscan doit afficher : /tmp/eicar.txt: Eicar-Test-Signature FOUND. Si vous voyez FOUND, votre antivirus fonctionne correctement.
